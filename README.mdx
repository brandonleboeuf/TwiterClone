# Social Media App from FreeCodeCamp.com

<!-- Link to video
https://www.youtube.com/watch?v=n1mdAPFq2Os -->

### TO START

  <!-- this starts React on localhost:3000 -->

cd socialMediaApp
npm run serve

  <!-- this starts the Apollo server on localhost:5000 -->

cd client
npm start

## TECH BEING USED:

- React
- Semantic UI
- GraphQL
- Express
- MongoDB (Atlas) (signed up with Brandon.LeBoeuf.Dev)
- Node.js
